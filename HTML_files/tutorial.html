Game Introduction

Basic

There are 81 unique cards in this SET game
Each card has four attributes: color, number, shape, and shading

Rule

Each set contains three cards that each attribute of the three cards are all the same or all different
One score will be awarded if a set is found

A set consists of three cards satisfying 4 conditions:
(1) They all have the same number or have three different numbers
(2) They all have the same symbol or have three different symbols
(3) They all have the same shading or have three different shadings
(4) They all have the same color or have three different colors

Goal

Find More SETs!
